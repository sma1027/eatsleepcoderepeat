make it work, make it right, make it fast

10 October 2014

Return matching elements from two arrays

Given  
a = [1, 2, 3]
b = [9, 4, 1]
# Return [1]

1 - make it work, 2- make it right

c = []  
a.each do |item1|
  b.each do |item2|
    if item1 == item2
      c << item1
    end
  end
end
c # [1]

3 - make it fast

a = [1, 2, 3]
b = [9, 4, 1]
a & b # [1]

The single & operator replaces 8 lines of code !!! Ruby is awesome !!!
