using devise to create users with admin credentials

02 March 2015

Using the devise gem to create users with admin credentials in Rails.

The setup

  1. Add devise:
    • Open your Gemfile and add gem 'devise'
  2. Open your terminal and run:
  • bundle install
  • rails g devise:install
  • Create the user table
    • rails g devise user
  • Add an admin column to the users table
    • rails generate migration add_admin_to_users admin:boolean
  • Execute migration
    • rake db:migrate

    • Helper methods from devise

    • user_signed_in?
    • current_user
    • current_user.admin?

    Check your schema for the current_user's attributes.
