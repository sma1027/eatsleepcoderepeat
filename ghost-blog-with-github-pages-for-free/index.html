ghost blog with github pages for free!!!

24 February 2015 on ghost, free, blog, github-pages, buster

The setup:

  1. Download node version 0.10.33(stable)
    • because Ghost requires Node.js 0.10.x (latest stable)
  2. Download ghost
    • unzip the folder $cd ghost $npm install --production $npm start
    • Go to 127.0.0.1:2368/ghost/ and setup ghost
  3. Install buster
    • $sudo easy_install pip
    • $pip install buster
  4. Setup Github Pages
    • Create a new repo on GitHub
    • cd ghost
    • $buster setup
    • enter repo name git@github.com:username/repo_name.git
    • add a custom domain name buster add-domain custom-domain-name.com so that it can be viewed at http://username.github.io/custom-domain-name
  5. Deploy!
    • buster generate --domain 127.0.0.1:2368
    • buster preview which can be viewed at localhost:9000
    • buster deploy
